# Savvy. | Savings made simple.

**Savings made simple.**

Savvy is a minimalist, voice-powered personal finance Operating System designed to help you track expenses, manage bills, and visualize your financial health. Built with a focus on aesthetics and usability, it combines powerful analytics with an organic, calming user experience.


## Key Features

### Smart Voice Assistant
Add transactions without typing. The AI recognizes context, categories, and transaction types automatically.
* **Say:** *"Spent 500 on petrol"* -> **Result:** Adds ₹500 to **Transport** (Expense).
* **Say:** *"Received salary 50000"* -> **Result:** Adds ₹50,000 to **Salary** (Income).
* **Say:** *"Paid rent 15000"* -> **Result:** Adds ₹15,000 to **Rent** (Bill).

### Powerful Analytics
* **Interactive Charts:**
    * **Line Chart:** Visualizes spending trends over time.
    * **Donut Chart:** Breaks down expenses by category (Food, Travel, etc.).
* **Smart Insights:** Automatically calculates **Daily Average Spending** and highlights your **Top Spending Category**.

### Budget Management
* Set a monthly budget cap.
* Visual progress bar animates to show usage.
* **Smart Alerts:** Get warned when you cross 80% of your budget or exceed the limit.

### Premium UI/UX
* **Glassmorphism Design:** Frosted glass cards on organic, floating blobs.
* **Micro-Interactions:** Smooth animations for list items (`slide-in`), charts (`progressive draw`), and numbers (`count-up`).
* **Responsive:** Fully optimized for Desktop, Tablet, and Mobile.

### Data Control
* **Filters:** Filter history by Date Range, Category, or Type (Income/Expense/Bill).
* **Export:** Download your data as `.csv` for Excel/Sheets.
* **Print:** Generate a clean PDF report of your transaction history.

## Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables, Animations, Flexbox/Grid).
* **Logic:** Vanilla JavaScript (ES6+ Modules).
* **Database & Auth:** Google Firebase (Firestore & Authentication).
* **Visualizations:** Chart.js.
* **Typography:** Google Fonts (**Zilla Slab**).
* **Icons:** FontAwesome 6.
